HEADER : thông tin về token và thuật toán mã hóa

  "alg": "HS256",
  "typ": "JWT"
}

  "name": "John Doe",
  "iat": 1516239022,
  "exp": 1516242622
}
Signature (Chữ ký) Được tạo bằng cách mã hóa header + payload + secret key
